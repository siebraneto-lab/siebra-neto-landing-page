# Siebra Neto | Compositor - Landing Page

🎵 Landing page oficial do compositor brasileiro **Siebra Neto**.

## Sobre

Site pessoal com estética dark/glassmorphism, efeitos neon e animações de áudio. Apresenta o trabalho musical de Siebra Neto, compositor brasileiro com músicas espirituais que transitam entre a força do rock, paz do reggae e poesia da MPB.

> "Jesus Cristo é o centro de toda a sua motivação musical."

## Tecnologias

- HTML5 semântico
- Tailwind CSS (via CDN)
- CSS3 com glassmorphism, neon glow e animações
- JavaScript vanilla
- Swiper.js (carrossel de lançamentos)
- Spotify Embed API
- Font Awesome Icons
- Google Fonts (Ubuntu + Inter)

## Funcionalidades

- 🎨 Design responsivo com estética dark/neon
- 🎵 Carrossel de lançamentos musicais com Swiper.js
- 🎧 Player do Spotify integrado (perfil do artista + faixa em destaque)
- ✨ Animações de equalizer, partículas flutuantes e ondas neon
- 📱 Menu mobile com navegação suave
- 🔗 Links para redes sociais e plataformas de streaming
